# Rozgar Radar - Skill Salary Map

A comprehensive platform for exploring job opportunities and salary insights across Indian states.

## Features

- **Interactive India Map** - Visualize job demand and salary information by district
- **Resume Analyzer** - Get ATS score, skill detection, and job match recommendations
- **Career Chatbot** - Get career guidance from EFOS Saathi chatbot
- **Filters & Analytics** - Advanced filtering and visual analytics with charts
- **Real-time Data** - District-wise demand and salary insights

## Tech Stack

- React 18 + TypeScript
- Vite for fast development
- Tailwind CSS for styling
- D3 & TopoJSON for mapping
- Recharts for data visualization
- GSAP for animations

## Local Development

## Features In Detail

### 1. Interactive Map
- Explore job opportunities by district
- Color-coded demand intensity
- Salary ranges and growth indicators

### 2. Resume Analyzer 📄
- Upload TXT, PDF, or DOCX files
- ATS Score calculation (Formatting, Keywords, Structure, Experience)
- Skill detection and matching
- Personalized job recommendations

### 3. Career Guidance 💬
- Integrated EFOS Saathi chatbot
- Real-time career counseling
- Multi-language support

### 4. Analytics Dashboard
- Demand growth trends
- Salary comparisons
- Skill distribution analysis
- Future predictions
